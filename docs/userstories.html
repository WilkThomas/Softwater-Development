1.0 User Stories

1.1.0 Filtering by Geographical Region

As a geological researcher, I want to track minerals in the produced water by geographical region to assist in my study of changes in soil composition caused by oil drilling in my area.

Elaboration: Users should be able to select specific minerals, so we need to support selection capability for minerals

Constraints: None (or some)

Effort Estimation: 5 person-hours

Acceptance Test: Perform minerals search operation in user interface, return with the format of table and figure

1.2.0 Filtering by Time Period

As an ecologist, I want to check the concentration of organic and inorganic contents in produced water so that I can assess the environmental impacts on surrounding ecosystem.

Elaboration: Users should be able to select multiple parameters like oil and grease, TOC, TDS, and pH, so we need to support multiple selections by time period

Constraints: None (or some)

Effort Estimation: 5 person-hours

Acceptance Test: Perform multiple search operation with the option of specific time period, return with the format of table and figure

1.3.0 Export GIS Map

As a government regulator, I want to obtain the statistical analysis of produced water quality information, including TDS, TSS, Na, Ca, SO4, and Cl, so that we can estimate future water quality and make more suitable regulation for produced water reuse.

Elaboration: Users should be able to provide statistical analysis, so we need to support statistical result for selected data.

Constraints: None (or some)

Effort Estimation: 6 person-hours

Acceptance Test:perform multiple parameters search operation in user interface, return with the GIS map

1.4.0 Customize Filter Value

As an Environmental Protection Agency member, I want to monitor the hazardous contents like heavy metals in oil basins, so that we can investigate the impact of oil drilling on ground water contamination.

Elaboration: Users should be able to filter the concentration of hazardous contents that higher than input value, so we need to support filtering

Constraints: Our database may not include all data needed

Effort Estimation: 4 person-hours

Acceptance Test: Perform a filter and return data that meet criteria

1.5.0 Generate Images and Tables

As an environmental researcher, I want to generate graphs and tables of produced water so that I can apply into my research paper on produced water treatment.

Elaboration: Users should be able to export the images and tables.

Constraints: None (or some)

Effort Estimation: 4 person-hours

Acceptance Test:Generate images and tables in standard formats

1.6.0 Filtering by Multiple Parameters

As an oil company executive, I want to evaluate the produced water quality so that I can select appropriate treatment technology to treat produced water before disposal.

Elaboration: Users should be able to select multiple parameters of water quality, so we need to support multiple selections.

Constraints: Part of data may not be provided in the database, so the result may miss some data

Effort Estimation: 4 person-hours

Acceptance Test: Perform a query from a user interface with multiple parameters, return a warning massage if data is unavailable
